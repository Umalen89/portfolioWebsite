Random Message Generator



  1. Description

    The objective of this project is to generate a random message based on the zodiac sing thematic. The message is divided on to 4 arrays, each of them represents a section of the sentence.

    1. Zodiac Sign
    2. Link between Zodiac sign and month related to the selcted sign.
    3. Dates corresponding to the Zodiac Sign
    4. End of sentence

    The link and End of sentences arrays have been given more than one option to choose from to randomize more the resulting sentence.

  2. How it works

    The program uses a function to create the random message using three variables (numSign - numLink - numEnd), numSign will store the random index of the Zodiac Sign and its correspondent dates of the month. numLink and numEnd as the name suggest will storage the random index number the link and end sentece respectively.
    When we are calling more than 1 time the function that generates the message, there is 2 more functions created with a global number history array both, to check if the next index number created has not been used in a previous sentece.
